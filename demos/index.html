These demos were rebuilt on the fix/layout-robustness branch. The main visible improvements: Stack/Grid/Container now apply layout CSS to the host (not the slot, which Shadow DOM drops silently), Tabs hide inactive panels via slot filtering, Text supports heading1/2/3 aliases, Table renders typed badge cells, and Metric accepts free-form trend strings.
